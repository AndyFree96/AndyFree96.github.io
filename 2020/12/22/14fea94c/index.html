Axios的响应对象

当我们发送一个请求给服务器后，它会返回一个响应。Axios的响应对象包含如下内容：

  • data - 服务器返回的响应主体数据
  • status - 服务器返回的响应状态码
  • statusText - 返回的状态信息
  • headers - 返回的头部信息
  • config - 请求的配置信息
  • request - 请求对象
