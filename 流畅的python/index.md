# 流畅的Python 第2版 (Luciano Ramalho)


[流畅的Python](https://book.douban.com/subject/36342440/)是在Python社区较为知名的一本书。它不是基础入门书，而是一本答疑解惑的书。为什么Python这样设计？哪些能力是语言提供的？哪些模式是社区长期沉淀出来的？它不会告诉你如何写一个`for`循环，也不会教你如何调用某个 API。它关注的是 Python 代码背后的运行机制：

- 对象如何协作？
- 抽象如何建立？
- 语言特性如何组合？

读完这本书最大的变化，不是记住了更多 Python API，而是开始用 Python 的方式思考问题。这也是高级 Python 开发者和普通 Python 使用者之间的重要区别。
